Checking your internet connection with speedtest-cli

August 4, 2012 One-minute read
Linux

Install speedtest-cli

sudo apt-get install python-pip
sudo pip install speedtest-cli

And now we test

speedtest-cli

If you wish to share you results just run with:

speedtest-cli --share
