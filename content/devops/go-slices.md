Slice is a useful data type that are used instead of arrays.

# Declaring a slice

```go
arr := [...]string{"a","b","c","d"}
```
Functions frequently used with slices

```go
len() - length
cap() - capacity
```

# Accessing slices
```go
fmt.Printf(s1[1])
```
